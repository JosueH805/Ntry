# Ntry — Bluetooth SaaS Access Control App

A mobile SaaS application that uses Bluetooth to communicate with an M5Stack 
microcontroller, enabling physical access control through a servo mechanism. 
Built as a junior design project at California Baptist University and selected 
for presentation at the CBU Design Showcase.


## Tech Stack
- **Frontend:** Flutter / Dart
- **Backend:** Firebase, Google Cloud Platform (GCP)
- **Hardware:** M5Stack Core3 (C++)
- **Communication:** Bluetooth Low Energy (BLE)
- **Project Management:** Jira, GitHub
- **Methodology:** Agile / Scrum

## Features
- Bluetooth communication between mobile app and M5Stack microcontroller
- Servo motor control triggered via mobile interface
- Cloud backend for data storage and authentication via Firebase
- Real-time device communication and status updates
